⚙️ Configuration

Connectez l'app à Google Calendar. Elle lira et analysera les événements de votre équipe selon la convention de nommage ci-dessous.

Convention de nommage des événements :
[P] TYPE Client – description

• [P] = travail pour le client (sans lui)  |  sans [P] = avec le client
• Types : PROP  REGIE  INTERNE
• Exemples : PROP Acme – démo  ·  [P] REGIE Orange – rapport  ·  INTERNE – réunion
