⚙️ Configuration initiale

Pour fonctionner de façon autonome, l'app a besoin d'un identifiant OAuth Google. C'est gratuit et prend ~5 minutes.

1
Allez sur console.cloud.google.com → créez un projet
2
Activez l'API Google Calendar
3
Credentials → + Créer → OAuth Client ID → Application Web → ajoutez http://localhost et l'URL de ce fichier en origines JS autorisées
4
Copiez le Client ID ici

Le Client ID est stocké localement dans votre navigateur. Vous ne le saisirez qu'une seule fois.
