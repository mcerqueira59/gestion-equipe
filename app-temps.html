⚙️ Configuration

Pour fonctionner de façon autonome, l'app a besoin d'un identifiant OAuth Google (gratuit, ~5 min).

1
Créez un projet sur console.cloud.google.com
2
Activez l'API Google Calendar
3
Credentials → + Créer → OAuth Client ID → Application Web → Origine JS : https://mcerqueira59.github.io
4
Copiez le Client ID ci-dessous

Le Client ID est stocké localement dans votre navigateur.

Charge équipe itroom
